# Jour 21 : Brouillage de mot de passe

## Partie 1

La tâche consiste à manipuler une chaîne de caractères en suivant une série d'instructions. Les opérations comprennent :

- `swap position X with position Y` : échange les lettres aux indices X et Y.
- `swap letter X with letter Y` : échange les lettres X et Y où qu'elles se trouvent.
- `rotate left/right X steps` : fait pivoter la chaîne entière.
- `rotate based on position of letter X` : fait pivoter la chaîne en fonction de l'indice d'une lettre spécifique. La rotation est de 1 + indice + (1 si l'indice >= 4) vers la droite.
- `reverse positions X through Y` : inverse la sous-chaîne entre les indices X et Y.
- `move position X to position Y` : déplace la lettre de l'indice X à l'indice Y.

En appliquant ces opérations dans l'ordre sur la chaîne `abcdefgh`, on obtient le mot de passe brouillé.

La solution pour la partie 1 est `aefgbcdh`.

## Partie 2

Pour la deuxième partie, nous devons inverser le processus. En partant du mot de passe brouillé `fbgdceah`, nous devons appliquer l'inverse de chaque opération dans l'ordre inverse pour retrouver le mot de passe d'origine.

- L'inverse de `swap` est `swap`.
- L'inverse de `rotate left` est `rotate right`, et vice-versa.
- L'inverse de `reverse` est `reverse`.
- L'inverse de `move position X to position Y` est `move position Y to position X`.
- L'inversion de `rotate based on position` est la plus complexe. Il faut déterminer l'indice d'origine qui aurait abouti à la position actuelle après la rotation. On peut tester chaque indice de départ possible pour trouver celui qui correspond.

En appliquant ces opérations inverses sur `fbgdceah`, on retrouve le mot de passe d'origine.

La solution pour la partie 2 est `bhdgfeac`.
