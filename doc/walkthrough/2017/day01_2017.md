# Day 1: Inverse Captcha (2017)

## Part 1

### Problem Description

We need to review a sequence of digits and find the sum of all digits that match the **next** digit in the list. The list is circular, so the digit after the last digit is the first digit.

### Approach

1.  **Parse Input**: Read the string of digits.
2.  **Iterate**: Loop through each digit at index `i`.
3.  **Compare**: Check if `digit[i]` equals `digit[(i + 1) % length]`.
4.  **Sum**: If they match, add the digit to the total sum.

### Solution

The solution for my input was **1150**.


## Part 2

### Problem Description

Now we need to consider the digit **halfway around** the circular list instead of the next digit. The list has an even number of elements.

### Approach

1.  **Calculate Step**: Determine the step size, which is `length // 2`.
2.  **Iterate**: Loop through each digit at index `i`.
3.  **Compare**: Check if `digit[i]` equals `digit[(i + step) % length]`.
4.  **Sum**: If they match, add the digit to the total sum.

### Solution

The solution for my input was **1064**.
